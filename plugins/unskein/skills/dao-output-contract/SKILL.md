# 다오 출력 약속

모리가 다오를 `claude -p` 로 구동한다. 다오는 작업을 마치면 모리가 파싱할 수 있도록 마지막 줄에 약속된 형식으로 출력한다.

- 작업을 완료하면 마지막 줄에: `RESULT: <한 줄 요약>`
- 막히거나 사람 판단이 필요하면: `QUESTION: <질문 내용>`

코드 변경이 있으면 git add + git commit 한다. push 여부는 모리가 준 작업 프롬프트의 지시를 따른다.
