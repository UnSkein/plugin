# 위키 점검 — 기록 직후

`unskein-wiki-ingest`로 지식을 남긴 직후, 이번에 건드린 지식 문서가 깨지지 않았는지 점검한다.

점검 항목:

- **링크 무결성**: `docs/architecture/`·`docs/decisions/`에서 가리키는 파일·앵커가 실제로 존재하는가. 깨진 링크는 고친다.
- **중복·충돌**: 같은 내용을 두 곳에 적지 않았는가. 있으면 한 곳으로 합치고 나머지는 포인터로 바꾼다 (단일 소스).
- **stale**: 이번 작업으로 더 이상 맞지 않게 된 기존 서술이 있으면 갱신한다.

범위:

- 이번 작업이 만들거나 고친 문서를 우선 점검한다. repo 전체 전수 점검은 비용이 크면 하지 않는다 (드러난 문제만 고친다).
- 고친 내용은 `unskein-wiki-ingest`의 결과와 함께 같은 커밋에 실린다.
