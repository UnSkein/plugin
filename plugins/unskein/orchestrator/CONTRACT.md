# 자식 에이전트 출력 규약

오케스트레이터가 자식 에이전트(다오)를 `claude -p` 로 구동한다. 자식은 작업을 마치면 오케스트레이터가 파싱할 수 있도록 마지막 줄에 약속된 형식으로 출력한다.

- 작업을 완료하면 마지막 줄에: `RESULT: <한 줄 요약>`
- 막히거나 사람 판단이 필요하면: `QUESTION: <질문 내용>`

코드 변경이 있으면 git add + git commit 한다. push 여부는 받은 작업 프롬프트의 지시를 따른다.

이 규약은 orchestrator/run_once.py 의 build_prompt() 가 자식 프롬프트에 직접 지시해 강제하고, extract_marker() 가 파싱한다 — 마커 문자열 변경 시 두 함수와 이 문서를 함께 고친다.
